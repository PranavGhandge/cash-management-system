# Cash Management System (CMS)

A desktop-based Cash Management System built using:

* React.js
* Tailwind CSS
* Electron.js
* Fastify
* MySQL

---

# Features

* Secure Login System
* Opening Balance Management
* Deposit & Withdrawal Transactions
* Bank Wise Reports
* Dashboard Analytics
* Auto Logout System
* Date Wise Transactions
* Professional Desktop Application

---

# Technologies Used

## Frontend

* React.js
* Tailwind CSS
* Recharts
* React Router DOM

## Backend

* Node.js
* Fastify
* MySQL

## Desktop App

* Electron.js

# Installation

## Clone Repository

```bash
git clone <repository-url>
```

---

# Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

---

# Backend Setup

```bash
cd backend

npm install

npm run dev
```

---

# Build Frontend

```bash
cd frontend

npm run build
```

---

# Build Backend EXE

```bash
cd backend

npm run build

npx pkg . --targets node18-win-x64
```

---

# Run Electron App

```bash
npm install

npm start
```

---

# Create Desktop EXE

```bash
npm run dist
```
